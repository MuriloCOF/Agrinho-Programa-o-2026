# Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente

Projeto desenvolvido para o Agrinho

Código em Java Script
## O que é o site?
Este é um site educativo desenvolvido para o Projeto Agrinho, com o tema "Agro Forte, Futuro Sustentável: equilíbrio entre produção e meio ambiente". O objetivo é mostrar, de forma acessível e interativa, como a agricultura pode ser produtiva e, ao mesmo tempo, respeitar a natureza.

O site apresenta conteúdos sobre os quatro pilares da sustentabilidade no campo: cuidado com o solo, preservação da água, uso de energia limpa e proteção da biodiversidade. Também explica práticas modernas como plantio direto, agricultura de precisão com drones, integração lavoura-pecuária-floresta e controle biológico de pragas — tudo sem depender de um servidor (back-end), funcionando apenas com HTML, CSS e JavaScript puro.

## Sobre o Jogo
No final da página, há um minijogo chamado "Colheita Sustentável". Nele, o jogador controla uma cesta (com o mouse, toque ou setas do teclado) e deve coletar sementes e gotas de água que caem do céu, ganhando pontos por cada item correto capturado. Ao mesmo tempo, precisa desviar ou não pegar lixo e veneno, que reduzem a pontuação ou tiram uma vida. O desafio dura 45 segundos e o jogador começa com 3 vidas. O recorde fica salvo no navegador, incentivando quem visita o site a tentar superar sua própria marca enquanto aprende, de forma lúdica, a importância de fazer escolhas conscientes no campo.

## Utilização da Inteligência Artificial
A inteligência artificial foi utilizada 98% como ferramenta de apoio no desenvolvimento deste projeto, auxiliando na criação do código, na organização do site e na implementação das funcionalidades. Todo o conteúdo foi revisado e adaptado para atender aos objetivos do projeto.
